# 词汇级语义相似度

词汇级的语义相似对于很多的以词为分析基础的场景而言是很重要的，比如信息抽取、歧义消解、同义词检测等。

词语相似度的计算方法有多种，这些方法可以分为两大类：

1. 基于大规模语料的统计分析，将词频和词分布作为计算词语相似度的依据，这种计算方法的结果依赖于选取的语料库，所以对于不同场景下的需求，需要准备不同的语料来提高效果。

2. 通过计算本体的概念之间的共性或者差异性来评估词语之间的相似性。

## 何为相似性

何为相似性？

看似是一个简单的问题，我们可能都是习以为常地讨论着相似性，但是却容易对这个基础的问题缺乏清晰的思考。

我们说相似，很容易想到的例子就是同义词，书与图书，未成年与未成年人，它们之间的相似性甚至可以是1。

另外一个我们常说的相似性是相关性，就是两个词背后的含义存在很强的关系，这种关系并不限制具体的关系，比如苹果和梨具有相关性，李娜与网球具有相关性，食物与水果的关系。

所以，当我们需要讨论相关性时，一定要明确我们所关心的相关性到底是啥，根据具体的需求来确定如何获取这种相关性。

因为基于统计方法是很难获得近义词这种信息的，但是另外一种相似性倒是可以通过统计方法获得。
